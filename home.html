Java

  • Introduction

    Introduction to Java

    Java is a high-level, class-based, object-oriented programming language designed to have as few implementation dependencies as possible.

    Syntax

    public class Main {
        public static void main(String[] args) {
            System.out.println("Hello, World!");
        }
    }

    Example

    The code above prints "Hello, World!" to the console.

    Problem

    Write a program to print "Java Programming" to the console.

    Answer

    public class Main {
        public static void main(String[] args) {
            System.out.println("Java Programming");
        }
    }

  • Variables

    Variables are containers for storing data values.

    Syntax

    int number = 10;
    String name = "John";

    Example

    Declare and print a variable:

    int age = 25;
    System.out.println("Age: " + age);

    Problem

    Declare a string variable to store your name and print it.

    Answer

    String myName = "Alice";
    System.out.println("Name: " + myName);

  • Loops

    Loops are used to execute a block of code repeatedly.

    Example: For Loop

    for (int i = 1; i <= 5; i++) {
        System.out.println(i);
    }

    Problem

    Write a program to print numbers from 1 to 10 using a while loop.

    Answer

    int i = 1;
    while (i <= 10) {
        System.out.println(i);
        i++;
    }

  • Object-Oriented Programming (OOP)

    OOP focuses on using objects to design and build applications.

    Example: Class and Object

    class Animal {
        String name;
    
        void speak() {
            System.out.println("Animal speaks");
        }
    }
    
    public class Main {
        public static void main(String[] args) {
            Animal dog = new Animal();
            dog.name = "Buddy";
            dog.speak();
        }
    }

    Problem

    Create a class "Car" with attributes "brand" and "speed" and a method to display these values.

    Answer

    class Car {
        String brand;
        int speed;
    
        void display() {
            System.out.println("Brand: " + brand + ", Speed: " + speed);
        }
    }
    
    public class Main {
        public static void main(String[] args) {
            Car car = new Car();
            car.brand = "Toyota";
            car.speed = 120;
            car.display();
        }
    }

    Strings in Java are objects that represent sequences of characters.

    String message = "Hello, World!";
                           System.out.println(message.toUpperCase());

    Problem

    Write a program to find the length of a string.

    Answer

    String str = "Java";
                           System.out.println("Length: " + str.length());

  • Inheritance

    Inheritance allows one class to acquire properties and methods of another class.

    Example

    class Parent {
        void display() {
            System.out.println("This is the parent class");
        }
    }
    
    class Child extends Parent {
        void show() {
            System.out.println("This is the child class");
        }
    }
    
    public class Main {
        public static void main(String[] args) {
            Child child = new Child();
            child.display();
            child.show();
        }
    }

    Problem

    Create a class hierarchy with "Animal" as a parent class and "Dog" as a child class. Add methods to each class and demonstrate inheritance.

  • Interfaces

    Interfaces in Java are used to achieve abstraction and multiple inheritance.

    Example

    interface Animal {
        void speak();
    }
    
    class Dog implements Animal {
        public void speak() {
            System.out.println("Woof!");
        }
    }
    
    public class Main {
        public static void main(String[] args) {
            Dog dog = new Dog();
            dog.speak();
        }
    }

  • Four Pillars of OOP

    • Encapsulation: Bundling data and methods.
    • Inheritance: Acquiring properties from a parent class.
    • Polymorphism: Multiple forms (method overloading and overriding).
    • Abstraction: Hiding implementation details.

  • Method Overriding

    Example

    class Parent {
        void display() {
            System.out.println("Parent class method");
        }
    }
    
    class Child extends Parent {
        @Override
        void display() {
            System.out.println("Child class method");
        }
    }
    
    public class Main {
        public static void main(String[] args) {
            Parent obj = new Child();
            obj.display();
        }
    }

  • Method Overloading

    Example

    class Calculator {
        int add(int a, int b) {
            return a + b;
        }
    
        double add(double a, double b) {
            return a + b;
        }
    }
    
    public class Main {
        public static void main(String[] args) {
            Calculator calc = new Calculator();
            System.out.println(calc.add(5, 10));
            System.out.println(calc.add(5.5, 10.5));
        }
    }

C Programming

  • Introduction to C

    C is a procedural programming language that provides low-level access to memory and is suitable for system programming.

    Syntax

    #include <stdio.h>
    
    int main() {
        printf("Hello, World!\n");
        return 0;
    }

    Example

    The code above prints "Hello, World!" to the console.

  • Loops in C

    Example: For Loop

    #include <stdio.h>
    
    int main() {
        for (int i = 1; i <= 5; i++) {
            printf("%d\n", i);
        }
        return 0;
    }

    This loop prints numbers from 1 to 5.

  • Functions in C

    Example

    #include <stdio.h>
    
    void greet() {
        printf("Hello, Welcome!\n");
    }
    
    int main() {
        greet();
        return 0;
    }

    This example defines and calls a function in C.

  • Pointers in C

    Example

    #include <stdio.h>
    
    int main() {
        int number = 10;
        int *ptr = &number;
        printf("Value: %d\n", *ptr);
        return 0;
    }

    This code demonstrates the use of pointers to access variable values.
